Projet realisé en C++ consistant à représenter un polynôme générique et effectuer diverses opérations sur ce polynôme :
-Evaluation d'un polynôme en une valeur donnée de x
-Calcul d'un  polynôme dérivé 
-Addition, soustraction et multiplication de deux polynômes via les opérateurs binaires +, -, *
-Implémentation des opérateurs +=, -=, *= entre deux polynômes ainsi que les opérateurs * et *= d’un polynôme par un scalaire
-Opération de division euclidienne d’un polynôme par un autre 
-Opération de composition de deux polynômes.


Fonctionnaltité avancée : 
- Ecriture d' un programme lisant une séquence simple de calculs élémentaires sur des polynômes et de les interpréter
